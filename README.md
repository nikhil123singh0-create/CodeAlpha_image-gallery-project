# CodeAlpha_image-gallery-project
# Premium Image Gallery

A modern and responsive Image Gallery built using HTML, CSS, and JavaScript.

## Features

- Responsive Design
- Category Filtering
- Fullscreen Lightbox
- Previous & Next Navigation
- Keyboard Controls
- Hover Animations
- Mobile Friendly Layout

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

```text
project/
│
├── index.html
├── style.css
├── script.js
└── images/
```

## How to Run

1. Download the project.
2. Open `index.html` in your browser.
